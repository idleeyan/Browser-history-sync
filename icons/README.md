# 图标文件

此文件夹存放扩展所需的图标文件。

## 所需图标规格

| 文件名 | 尺寸 | 用途 |
|--------|------|------|
| icon16.png | 16x16 | 扩展页面图标、工具栏图标 |
| icon48.png | 48x48 | 扩展管理页面图标 |
| icon128.png | 128x128 | Chrome Web Store 图标 |

## 图标设计建议

- 使用 PNG 格式，支持透明背景
- 建议使用简洁的矢量风格图标
- 主色调建议：蓝色系（#3B82F6）或绿色系（#10B981）
- 图标主题建议：同步、历史记录、云存储相关元素

## 临时解决方案

在开发阶段，可以使用在线图标生成工具创建临时图标：
1. [Chrome Extension Icon Generator](https://chrome-extension-icon-generator.com/)
2. 或使用任意 128x128 图片，通过工具生成其他尺寸

**注意**：发布到 Chrome Web Store 前，请替换为正式设计的图标。
